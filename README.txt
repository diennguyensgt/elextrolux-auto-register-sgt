HƯỚNG DẪN SỬ DỤNG ELEXTROLUX AUTO REGISTER

1. Yêu cầu hệ thống:
   - Windows 10 trở lên
   - Google Chrome đã cài đặt
   - Kết nối internet
   - Microsoft Excel

2. Cách sử dụng:
   a. Chuẩn bị file Excel:
      - Mở file "danh_sach_san_pham_mau.xlsx"
      - Điền thông tin sản phẩm theo mẫu
      - Lưu file với tên khác (ví dụ: "danh_sach_san_pham.xlsx")

   b. Chạy chương trình:
      - Chạy file "auto_register.exe"
      - Chương trình sẽ tự động mở Chrome và bắt đầu đăng ký
      - Làm theo hướng dẫn trên màn hình

3. Cấu trúc file Excel:
   - Cột A: URL sản phẩm
   - Cột B: Email
   - Cột C: Tên
   - Cột D: Họ
   - Cột E: Số điện thoại
   - Cột F: Số serial

4. Lưu ý:
   - Đảm bảo file Excel không bị khóa
   - Không đóng Chrome trong quá trình chạy
   - Nếu gặp lỗi, vui lòng kiểm tra:
     + Kết nối internet
     + Định dạng file Excel
     + Chrome đã cài đặt đúng cách
